# Neo4j Skills

Neo4j Skills for Coding and other Agents including Cypher skills and others.

## What are Skills?

Skills are modular, reusable capabilities that can be used by AI agents and coding assistants to perform specific tasks. They provide a standardized way to define and share functionality across different AI systems.

This repository contains a collection of skills specifically designed for working with Neo4j graph databases, including:

- **Cypher Skills**: Skills for writing, optimizing, and executing Cypher queries
- **Migration Skills**: Skills for migrating data to and from Neo4j
