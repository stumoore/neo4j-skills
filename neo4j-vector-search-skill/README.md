# neo4j-vector-search-skill

Guides agents through vector search in Neo4j: creating vector indexes, storing embeddings on nodes, using the SEARCH clause (2026.02.1+) or db.index.vector.queryNodes() procedure, and choosing embedding dimensions.

**Install:**
```bash
npx skills add https://github.com/neo4j-contrib/neo4j-skills --skill neo4j-vector-search-skill
```

Or paste this link into your coding assistant:
https://github.com/neo4j-contrib/neo4j-skills/tree/main/neo4j-vector-search-skill
