# neo4j-import-skill

Guides agents through importing structured data into Neo4j: LOAD CSV, batch upserts with CALL IN TRANSACTIONS, and offline bulk load with neo4j-admin import.

**Install:**
```bash
npx skills add https://github.com/neo4j-contrib/neo4j-skills --skill neo4j-import-skill
```

Or paste this link into your coding assistant:
https://github.com/neo4j-contrib/neo4j-skills/tree/main/neo4j-import-skill
