# neo4j-mcp-skill

Installs and configures the official Neo4j MCP server so AI agents can connect to Neo4j via Claude Code, Claude Desktop, Cursor, Windsurf, VS Code, Kiro, and other MCP-compatible editors. Covers stdio and HTTP transport, read-only mode, and the four available tools.

**Install:**
```bash
npx skills add https://github.com/neo4j-contrib/neo4j-skills --skill neo4j-mcp-skill
```

Or paste this link into your coding assistant:
https://github.com/neo4j-contrib/neo4j-skills/tree/main/neo4j-mcp-skill
