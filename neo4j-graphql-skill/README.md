# neo4j-graphql-skill

Guides agents through building a GraphQL API on Neo4j with the Neo4j GraphQL Library: type definitions, @relationship, @cypher directive, @authorization, and OGM usage.

**Install:**
```bash
npx skills add https://github.com/neo4j-contrib/neo4j-skills --skill neo4j-graphql-skill
```

Or paste this link into your coding assistant:
https://github.com/neo4j-contrib/neo4j-skills/tree/main/neo4j-graphql-skill
