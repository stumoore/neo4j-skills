# Cypher Skills

This directory contains skills for working with Cypher queries in Neo4j.

## About Cypher

Cypher is Neo4j's graph query language that allows you to store and retrieve data from the graph database. It is a declarative, SQL-inspired language for describing visual patterns in graphs using ASCII-Art syntax.
