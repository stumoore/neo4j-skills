# neo4j-gds-skill

Guides agents through Graph Data Science: graph projection, algorithm execution (PageRank, Louvain, FastRP, shortest paths), memory estimation, and the GDS Python client.

> **Note:** GDS is available on Aura Professional and Enterprise only — not Aura Free.

**Install:**
```bash
npx skills add https://github.com/neo4j-contrib/neo4j-skills --skill neo4j-gds-skill
```

Or paste this link into your coding assistant:
https://github.com/neo4j-contrib/neo4j-skills/tree/main/neo4j-gds-skill
