# neo4j-spring-data-skill

Guides agents through Spring Data Neo4j: @Node entity mapping, @Relationship, Neo4jRepository, @Query annotations, reactive repositories, and Spring AI Neo4jVectorStore.

**Install:**
```bash
npx skills add https://github.com/neo4j-contrib/neo4j-skills --skill neo4j-spring-data-skill
```

Or paste this link into your coding assistant:
https://github.com/neo4j-contrib/neo4j-skills/tree/main/neo4j-spring-data-skill
