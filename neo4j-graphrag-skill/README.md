# neo4j-graphrag-skill

Guides agents through GraphRAG pipelines on Neo4j: retriever selection (HybridCypherRetriever, VectorCypherRetriever), retrieval_query graph traversal patterns, SimpleKGPipeline for knowledge graph construction, and LangChain/LlamaIndex integrations.

**Install:**
```bash
npx skills add https://github.com/neo4j-contrib/neo4j-skills --skill neo4j-graphrag-skill
```

Or paste this link into your coding assistant:
https://github.com/neo4j-contrib/neo4j-skills/tree/main/neo4j-graphrag-skill
